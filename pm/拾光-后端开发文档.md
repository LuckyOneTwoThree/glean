## 1. 架构总览与核心决策

### 1.1 核心决策：方案B — 保留前端现有schema

**为什么不用方案A（按PRD重做schema）？**

| 考量 | 方案A（PRD规范化） | 方案B（保留现有+扩展） |
|------|-------------------|----------------------|
| 前端改动量 | 全部Model/Service/Provider/Screen重写 | 仅补全TODO和新增功能 |
| 数据迁移 | 需要v1→v2迁移脚本，风险高 | 无需迁移，增量升级 |
| 开发周期 | 2-3周 | 1-2周 |
| 风险 | 前端所有页面可能break | 最小化风险 |
| PRD对齐 | 完美对齐 | 功能对齐，结构差异可接受 |

**结论**：前端已有完整的 `database_service.dart` + 7个Model + 7个Service + Provider，功能逻辑基本正确。选择方案B，在现有基础上补全TODO、修复问题、扩展缺失功能。

### 1.3 Schema差异对照（前端 vs PRD）

| 前端表名 | PRD表名 | 差异说明 | 处理方式 |
|----------|---------|---------|---------|
| `articles` | `articles` | 前端id=TEXT,内嵌score/summary; PRD id=INTEGER,拆分analysis/summaries | **保留前端**，内嵌更简单 |
| `feeds` | `sources` | 前端缺`is_domestic`、`credibility`(0-10)、`status` | **扩展feeds表**，新增字段 |
| `briefings` | `reports`+`report_items` | 前端单表无分组; PRD两表支持分组 | **扩展briefings表**，新增分组字段 |
| `scores` | `analysis` | 字段名不同（credibility vs score_source） | **保留scores**，重命名对齐 |
| 无 | `summaries` | 前端摘要内嵌articles | **保留内嵌**，无需独立表 |
| `user_config` | `user_config` | 前端单行多列; PRD key-value | **保留单行多列**，更直观 |
| `llm_config` | user_config中 | 前端独立表; PRD在user_config中 | **保留独立表**，职责更清晰 |
| `execution_logs` | `execution_logs` | 字段名不同 | **保留前端字段** |
| 无 | `article_tags` | 前端无 | **新增表** |
| 无 | `llm_costs` | 前端无 | **新增表** |

```dart
// database_service.dart 改造
static const _dbVersion = 2; // 从1升级到2

Future<void> _onCreate(Database db, int version) async {
  // v1 原始6张表
  await db.execute('''CREATE TABLE articles (...)''');  // 原样
  await db.execute('''CREATE TABLE feeds (...)''');     // 原样 + 新增字段
  await db.execute('''CREATE TABLE briefings (...)'''); // 原样 + 新增字段
  await db.execute('''CREATE TABLE scores (...)''');    // 原样
  await db.execute('''CREATE TABLE user_config (...)''');// 原样

  // v2 新增表
  await db.execute('''CREATE TABLE article_tags (...)''');
  await db.execute('''CREATE TABLE llm_costs (...)''');
  // 索引
  // ...
  // 种子数据
  await _seedPresetFeeds(db);
  await _seedDefaultConfig(db);
}

Future<void> _onUpgrade(Database db, int oldVersion, int newVersion) async {
  if (oldVersion < 2) {
    // v1 → v2 迁移
    await db.execute('ALTER TABLE feeds ADD COLUMN is_domestic INTEGER DEFAULT 1');
    await db.execute('ALTER TABLE feeds ADD COLUMN credibility REAL DEFAULT 5.0');
    await db.execute('ALTER TABLE feeds ADD COLUMN status TEXT DEFAULT "active"');
    await db.execute('ALTER TABLE briefings ADD COLUMN total_fetched INTEGER DEFAULT 0');
    await db.execute('ALTER TABLE briefings ADD COLUMN domestic_count INTEGER DEFAULT 0');
    await db.execute('ALTER TABLE briefings ADD COLUMN international_count INTEGER DEFAULT 0');
    await db.execute('ALTER TABLE briefings ADD COLUMN categories_json TEXT');
    await db.execute('ALTER TABLE llm_config ADD COLUMN timeout INTEGER DEFAULT 30');
    await db.execute('ALTER TABLE llm_config ADD COLUMN budget_tokens INTEGER DEFAULT 10000');
    await db.execute('''CREATE TABLE article_tags (...)''');
    await db.execute('''CREATE TABLE llm_costs (...)''');
    // 为已有feeds设置is_domestic和credibility
    await _migrateFeedsMetadata(db);
    // 插入预置数据源（如果feeds表为空）
    await _seedPresetFeedsIfEmpty(db);
  }
}

/// v1→v2迁移：为已有feeds补充元数据
Future<void> _migrateFeedsMetadata(Database db) async {
  final feeds = await db.query('feeds');
  for (final feed in feeds) {
    final url = feed['url'] as String? ?? '';
    final isDomestic = _guessIsDomestic(url);
    final credibility = feed['credibility_weight'] != null
        ? (feed['credibility_weight'] as double) * 2 // 旧0-5映射到0-10
        : 5.0;
    await db.update(
      'feeds',
      {
        'is_domestic': isDomestic ? 1 : 0,
        'credibility': credibility,
        'status': 'active',
      },
      where: 'id = ?',
      whereArgs: [feed['id']],
    );
  }
}

/// 根据URL域名判断是否国内源
bool _guessIsDomestic(String url) {
  final domesticDomains = [
    'qbitai.com', '36kr.com', 'infoq.cn', 'ifanr.com',
    'sspai.com', 'ithome.com', 'jiqizhixin.com', 'huxiu.com',
  ];
  return domesticDomains.any((d) => url.contains(d));
}

/// v1→v2迁移：如果feeds表为空则插入预置源
Future<void> _seedPresetFeedsIfEmpty(Database db) async {
  final count = Sqflite.firstIntValue(
    await db.rawQuery('SELECT COUNT(*) FROM feeds'),
  );
  if (count == 0) {
    await _seedPresetFeeds(db);
  }
}
```

```dart
Future<List<Article>> _fetchFromFeed(Feed feed) async {
  final dio = Dio();
  dio.options.connectTimeout = const Duration(seconds: 30);
  dio.options.receiveTimeout = const Duration(seconds: 30);
  dio.options.headers = {
    'User-Agent': 'Glean/1.0 (RSS Reader)',
  };

  // 1. HTTP GET 获取RSS XML
  final response = await dio.get<String>(feed.url);
  if (response.data == null || response.data!.isEmpty) return [];

  // 2. 解析XML
  final xmlStr = response.data!;
  final items = _parseRssXml(xmlStr);

  // 3. 转换为Article对象
  final now = DateTime.now().millisecondsSinceEpoch;
  return items.map((item) => Article(
    id: 'a${now}_${item.link.hashCode.abs()}',
    title: item.title ?? '无标题',
    url: item.link ?? '',
    content: item.content ?? item.description,
    summaryOne: item.description,
    sourceName: feed.name,
    sourceUrl: feed.url,
    category: feed.category,
    publishedAt: item.pubDate ?? now,
    fetchedAt: now,
    scoreCredibility: feed.credibility / 2, // 初始值，后续评分覆盖
  )).where((a) => a.url.isNotEmpty).toList();
}
```

**RSS解析方案**：

使用 `xml` 包手动解析（比 `webfeed` 更灵活，支持RSS 2.0 + Atom 1.0）：

```dart
List<_RssItem> _parseRssXml(String xmlStr) {
  final document = XmlDocument.parse(xmlStr);
  final items = <_RssItem>[];

  // RSS 2.0: <rss><channel><item>
  final rssItems = document.findAllElements('item');
  if (rssItems.isNotEmpty) {
    for (final item in rssItems) {
      items.add(_RssItem(
        title: _getElementText(item, 'title'),
        link: _getElementText(item, 'link'),
        description: _getElementText(item, 'description'),
        content: _getElementText(item, 'content:encoded') ?? _getElementText(item, 'description'),
        pubDate: _parseDate(_getElementText(item, 'pubDate')),
        author: _getElementText(item, 'dc:creator') ?? _getElementText(item, 'author'),
      ));
    }
    return items;
  }

  // Atom 1.0: <feed><entry>
  final atomEntries = document.findAllElements('entry');
  for (final entry in atomEntries) {
    final linkEl = entry.findElements('link').firstWhere(
      (el) => el.getAttribute('rel') == 'alternate' || el.getAttribute('href') != null,
      orElse: () => XmlElement(XmlName('link')),
    );
    items.add(_RssItem(
      title: _getElementText(entry, 'title'),
      link: linkEl.getAttribute('href') ?? '',
      description: _getElementText(entry, 'summary'),
      content: _getElementText(entry, 'content'),
      pubDate: _parseDate(_getElementText(entry, 'published') ?? _getElementText(entry, 'updated')),
      author: _getElementText(entry, 'author/name'),
    ));
  }

  return items;
}
```

### 6.2 去重算法修正

**当前问题**：
- Jaccard用字符级分词（`a.split('')`），应改为词级
- 阈值0.85，PRD要求0.75
- 综合相似度公式：PRD = 0.6×Jaccard + 0.4×编辑距离归一化

**修正后**：

```dart
Future<bool> _checkDuplicate(String title, String url) async {
  // 步骤1：URL精确匹配
  final urlMatch = await _db.query('articles', where: 'url = ?', whereArgs: [url]);
  if (urlMatch.isNotEmpty) return true;

  // 步骤2：相似度去重（只比较最近7天的文章）
  final weekAgo = DateTime.now().subtract(const Duration(days: 7)).millisecondsSinceEpoch;
  final recentArticles = await _db.query(
    'articles',
    where: 'fetched_at >= ?',
    whereArgs: [weekAgo],
    orderBy: 'fetched_at DESC',
    limit: 500,
  );

  for (final map in recentArticles) {
    final existingTitle = map['title'] as String;
    final jaccard = _wordJaccardSimilarity(title, existingTitle);
    final editSim = _editDistanceSimilarity(title, existingTitle);
    final combined = 0.6 * jaccard + 0.4 * editSim;
    if (combined >= 0.75) return true;
  }

  return false;
}

/// 词级Jaccard相似度（中文按2-gram分词，英文按空格分词）
double _wordJaccardSimilarity(String a, String b) {
  final setA = _tokenize(a).toSet();
  final setB = _tokenize(b).toSet();
  if (setA.isEmpty && setB.isEmpty) return 1.0;
  final intersection = setA.intersection(setB).length;
  final union = setA.union(setB).length;
  return union == 0 ? 0 : intersection / union;
}

/// 简易分词：中文2-gram + 英文按空格
List<String> _tokenize(String text) {
  final tokens = <String>[];
  // 英文按空格分词
  final words = text.split(RegExp(r'\s+'));
  tokens.addAll(words.where((w) => w.length > 1));
  // 中文2-gram
  for (var i = 0; i < text.length - 1; i++) {
    if (RegExp(r'[\u4e00-\u9fff]').hasMatch(text.substring(i, i + 1))) {
      tokens.add(text.substring(i, i + 2));
    }
  }
  return tokens;
}
```

```dart
Future<Score> _scoreWithRules(Article article) async {
  // 来源可信度：从Feed表查询
  double scoreSource = 5.0; // 默认值
  try {
    final feedMaps = await _db.query('feeds',
      where: 'url = ?', whereArgs: [article.sourceUrl]);
    if (feedMaps.isNotEmpty) {
      scoreSource = (feedMaps.first['credibility'] as num?)?.toDouble() ?? 5.0;
    }
  } catch (_) {}

  // 信息密度：基于规则打分（0-10）
  double scoreDensity = 0;
  final content = article.content ?? article.summaryOne ?? '';

  if (content.length > 500) scoreDensity += 2;
  if (content.length > 1500) scoreDensity += 2;
  if (_hasNumbers(content)) scoreDensity += 1;
  if (_hasLinks(content)) scoreDensity += 1;
  if (_hasImages(content)) scoreDensity += 1;
  if (_isSpecificTitle(article.title)) scoreDensity += 1;
  if (_matchesFocusDomains(article.category)) scoreDensity += 2;
  scoreDensity = scoreDensity.clamp(0, 10);

  final scoreOverall = scoreSource * 0.5 + scoreDensity * 0.5;

  return Score(
    id: 's${DateTime.now().millisecondsSinceEpoch}',
    articleId: article.id,
    mode: 'local',
    credibility: scoreSource,
    density: scoreDensity,
    total: scoreOverall,
    scoredAt: DateTime.now().millisecondsSinceEpoch,
  );
}

bool _hasLinks(String text) => RegExp(r'https?://').hasMatch(text);
bool _hasImages(String text) => RegExp(r'<img|!\[').hasMatch(text);
bool _isSpecificTitle(String title) => title.length > 10 && !RegExp(r'^(今日|本周|最新|重要)').hasMatch(title);
```

```dart
String _extractOneLine(Article article) {
  // 优先使用RSS description（≤80字直接用）
  if (article.summaryOne != null && article.summaryOne!.isNotEmpty) {
    final desc = article.summaryOne!;
    return desc.length <= 80 ? desc : '${desc.substring(0, 77)}...';
  }
  // 否则取content前100字截断到完整句
  final content = article.content ?? '';
  if (content.isEmpty) return article.title;
  final firstSentence = content.split(RegExp(r'[。！？.!?]')).first;
  if (firstSentence.length <= 80) return firstSentence;
  return '${firstSentence.substring(0, 77)}...';
}

/// 三要点：取content前3个完整句
List<String> _extractPoints(Article article) {
  if (article.summaryPoints.isNotEmpty) return article.summaryPoints;
  final content = article.content ?? '';
  if (content.isEmpty) return ['暂无要点'];

  final sentences = content
      .split(RegExp(r'[。！？.!?]'))
      .where((s) => s.trim().length > 5)
      .map((s) => s.trim().length > 50 ? '${s.trim().substring(0, 47)}...' : s.trim())
      .take(3)
      .toList();

  if (sentences.isEmpty) return ['暂无要点'];
  while (sentences.length < 3) sentences.add('');
  return sentences;
}
```

## 7. Provider层对接

### 7.1 articlesProvider改造

```dart
final articlesProvider = FutureProvider<List<Map<String, dynamic>>>((ref) async {
  if (kIsWeb) {
    // Web保留mock数据
    return _getMockArticles();
  }

  final db = ref.watch(databaseServiceProvider);
  final filter = ref.watch(homeFilterProvider);
  final sort = ref.watch(homeSortProvider);
  final favoritedIds = ref.watch(favoritedArticleIdsProvider);

  String? where;
  List<Object?>? whereArgs;

  switch (filter) {
    case 'unread':
      where = 'is_read = 0';
      break;
    case 'favorited':
      where = 'is_favorited = 1';
      break;
  }

  final orderBy = sort == 'score' ? 'score_total DESC' : 'published_at DESC';

  final results = await db.query('articles', where: where, whereArgs: whereArgs, orderBy: orderBy);

  if (results.isEmpty) return _getMockArticles();

  // 同步收藏状态
  return results.map((map) {
    final id = map['id'] as String;
    return Map<String, dynamic>.from(map)
      ..['is_favorited'] = favoritedIds.contains(id) ? 1 : (map['is_favorited'] ?? 0);
  }).toList();
});
```
