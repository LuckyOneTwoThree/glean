在今天凌晨举行的 OpenAI 开发者闭门会议上，首席技术官 Mira Murati 展示了名为 GPT-Next 的全新模型。这款模型并不单纯是 GPT-4 的放大版，而是基于一种被内部称为“深思熟虑”的全新推理架构。

GPT-Next 在解决高等数学问题时表现出了惊人的毅力。在传统的测试中，模型往往会因为逻辑跳跃而产生错误，但 GPT-Next 会在内部生成数千个候选步骤，并通过强化学习自主发现并纠正其中的错误。

“我们不再只是让机器预测下一个字，” Murati 说道，“我们在教它如何思考，如何在给出答案之前进行‘系统性推理’。” 这一转变意味着 AI 在法律咨询、科学研究等对严谨性要求极高的领域将有质的突破。

“这不仅仅是参数的增加，这是范式的转移。我们将迎来一个能够自主修正逻辑谬误的 AI 时代。” —— 行业分析师

目前，该模型已对部分企业级用户开放 API 预览版，预计在下个季度正式推送给所有 Plus 用户。
